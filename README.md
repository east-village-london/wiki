# East Village London Wiki

A community-maintained knowledge base for residents of [East Village, London](https://en.wikipedia.org/wiki/East_Village,_London).

**Site:** [wiki.eastvillage.london](https://wiki.eastvillage.london)

## Contributing

The wiki is built with [Zensical](https://zensical.org) and deployed via GitHub Pages.

Content lives in the [docs/](docs/) directory as Markdown files. To add or edit a page:

1. Fork the repository
2. Edit or create a Markdown file under `docs/`
3. Update the `nav` in [zensical.toml](zensical.toml) if adding a new page
4. Open a pull request

Keep entries factual, specific to East Village London where relevant, and written for someone encountering the problem for the first time.

## Local development

```
pip install zensical
zensical serve
```
